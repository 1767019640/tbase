# tbase

Baselines of trade agents use reinforcement learning algorithms.

支持环境： python3(>=3.5)

# 安装

[docker install](https://docs.docker.com/install/)

# Features

- GPU 加速
- 支持多进程方式运行 Enviroment
- 支持单Agent与多Agent, 例如: MADDPG
- 通过运行参数选择:

  - 环境
  - 算法
  - Policy-Net
  - Value-Net
